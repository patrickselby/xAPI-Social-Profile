The AcrossX verbs will be available here: http://purl.org/xapi/acrossx/verbs


A quick check I found only several verbs in Social vocabulary table are not included in there: link, save, upload, download, delete, moderate, applied(LinkedIn?). (could use "responded" for "reply", responded is broader)

"Created" is included in Dr Kitto's paper, they used ActivityStream's vocab, that's broken, many AS vocabs are broken. So Jason created new ones in purl.

Furthermore, we really need to see verbs' use cases to know if one can be replaced by another. For example, if "created" means creating a discussion topic or asking a question on FB or a forum, then we can use "posted" and "asked", both already existed.
